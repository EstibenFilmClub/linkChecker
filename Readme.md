# Run with Docker 

##  Dependencies in your OS
- docker
- docker-compose

## Install

Go to the root of the project, and typing:
```
docker-compose up 
```
Stop docker 
```
docker-compose down
```

# Run with Python

## To run the script you have to posicionate in the directory of the enviroment and run the script

```python index.py```

## the dependencies installed are

- pip install flet
